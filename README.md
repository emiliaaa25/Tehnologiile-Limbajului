Recenzie Radar — React + Vite scaffold

Setup

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

Notes

- Static backup of the previous prototype is available at `index.static.html`.
- The mock dataset is in `public/aspects_stats.json`.
- Styles are in `src/styles.css` (which imports the legacy top-level `styles.css`).
